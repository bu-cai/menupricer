# 第六部分：产品设计 ＋ 第七部分：商业模式

# 六、产品设计（以Top3产品为例）

## 6.1 「薪算盘」— 五险一金税后工资计算器

### 版本规划

| 层级 | 功能 | 定价 |
|---|---|---|
| **MVP（第一版必须上线）** | ①月薪→到手工资计算（个税+五险一金）②城市选择（先北上广深杭+成都等10城）③2026年最新基数/比例 ④结果明细表（公司成本/个人扣除逐项）⑤移动端适配 ⑥每城市独立URL（pSEO基础） | 免费 |
| 免费版补充（V1.1-1.3） | 年终奖计税对比、年度汇算估算、40+城市、offer对比（两份薪资并排） | 免费 |
| **专业版（个人会员）** | 结果导出PDF/图片、历史记录、社保断缴影响模拟、跳槽涨薪需求测算（「到手不变需要谈多少」） | ¥19/年 或 ¥3单次导出 |
| **企业版（HR）** | Excel批量核算（整表员工）、企业成本测算、API接入、定制报告白标 | ¥299-999/年 |
| 未来功能 | 全国社平工资数据库、个税专项扣除向导（12月脉冲）、灵活就业参保档位对比、薪酬报告（数据积累后） | — |

**第一版必须上线的理由**：计算准确+城市页+移动端是获取SEO信任的最小闭环；分享图（「我的工资单明细」截图美化）是社交传播钩子，第一版就要做。

### 关键设计细节
- 结果页默认生成「可分享长图」：工资构成瀑布图＋网站水印——小红书/朋友圈自传播。
- 每个城市页底部放「本市2026年社保基数政策原文摘要+更新日期」——E-A-T信号，同时是GEO引用素材。
- Schema：`FAQPage` +自研计算器说明；页面标题模式「2026{城市}五险一金计算器 - 月薪{X}到手多少」。

## 6.4 通用产品设计原则（所有工具）

1. **三秒可用**：无注册墙即可用核心功能；注册只为保存历史。
2. **付费点后置且单点**：卡在「导出/批量/高清/无水印」，绝不卡核心计算/处理（迅捷式付费墙=口碑自杀）。
3. **每个工具页=着陆页**：H1含关键词、首屏工具、下方「怎么用+原理+FAQ」2000字内容区（吃知识词+GEO）。
4. **分享即拉新**：所有结果支持生成带二维码水印的分享图。
5. **双端**：网页(SEO主体)+微信小程序(支付与留存主体)，同名同品牌。

# 七、商业模式分析（中国市场）

## 7.1 十种模式适配度评估

| 模式 | 中国市场现实 | 适合工具站吗 | 建议 |
|---|---|---|---|
| 纯免费+广告 | 百度联盟eCPM低（¥1-5/千次），Adsense稍好但需外币收款；工具页广告位有限 | 仅适合巨量流量型（日均PV>10万） | 兜底手段，不当主模式 |
| **Freemium** | 已被稿定/ProcessOn/草料充分验证；免费获客+5%以内付费率 | ✅最适合 | **主模式**：核心免费+增值付费 |
| 会员订阅 | C端订阅疲劳（会员太多）；月订阅退订率高，**年费+促销价**是主流（¥99以下决策快） | ✅适合专业人群工具 | 对运营/教师/HR等职业用户用年费 |
| **按次付费（微信支付）** | 中国特色优势：微信支付零摩擦，¥1-9.9冲动付费极顺畅；乐棋证件照/Bigjpg验证 | ✅✅C端最优 | 考试/应急场景用单次，转化率高于订阅 |
| API收费 | TinyPNG/聚合数据模式；开发者客户稳定但获客慢 | ✅适合数据型工具 | 节假日API/社保数据API/违禁词API做第二曲线 |
| 企业版 | 中国B端要发票、要私有化、决策慢，但客单价高且稳定 | ✅成熟后再做 | 流量起来后自然有企业询价，被动承接 |
| 广告联盟/CPS | 金融（贷款/保险）、法律咨询、快递、电商CPS单价高 | ✅适合计算器类 | 房贷计算器×贷款CPS、赔偿计算器×律师咨询 |
| 模板商城 | 稿定/锤子简历验证；需持续生产模板 | 视品类 | 简历/PPT类可做，抽成或买断 |
| 积分制 | 签到/分享得次数——拉新裂变有效但记账复杂 | 部分适合 | AI类工具用「次数包」替代积分更简单 |
| 按次收费(线下服务撮合) | 冲印/打印/代办 | 补充 | 报名照→冲印邮寄是自然延伸 |

## 7.2 推荐的组合模型

```
流量层（免费）        变现层（分人群）
─────────────       ─────────────────────────
调休日历/退休计算器 →  广告+联盟CPS（金融/法律/快递）
报名照/图片处理    →  单次微支付 ¥3-9.9（C端应急）
AIGC/违禁词/评语   →  订阅 ¥29-99/年（职业用户）
数据型（节假日/社保）→  API包月（开发者/企业）
所有产品           →  企业版被动承接（发票+批量）
```

## 7.3 定价心理学（中国用户）

| 原则 | 说明 |
|---|---|
| ¥9.9锚点 | C端单次付费甜蜜点：¥1-3无感、¥9.9需给足理由、>¥19.9需强场景（毕业/求职） |
| 年费打折心智 | 「¥19/月 或 ¥99/年」——绝大多数选年费，现金流前置 |
| 首单1元 | 微信支付首单¥0.99体验价，转化率可翻倍 |
| 拒绝自动续费默认勾选 | 短期收入+长期口碑毁灭（黑猫投诉重灾区），做「到期提醒手动续」反而成知乎口碑素材 |
| 发票能力 | 个体户/小微企业主体+电子普票，解锁B端订单 |
