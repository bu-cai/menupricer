# 第三部分：50个蓝海方向（低竞争 × 个人可做 × 有付费空间）

> 筛选标准：搜索需求稳定 / 竞争相对低 / 开发·维护·AI成本低 / 1人可完成 / 高频使用 / 有付费点 / 适合SEO+知乎+小红书+公众号+B站。
> 评分维度：需求=搜索量与频次；竞争=现有产品强度(★少=竞争低=好)；难度=开发工作量(★少=简单)；商业=变现空间；SEO=自然流量潜力；GEO=被AI搜索引用潜力。推荐指数为综合。
> 已剔除违法/侵权/灰色方向。

## 组合策略建议

不要做50个孤岛，按「**1个主域名 + 3~4条产品线**」组合，权重互养：

| 组合 | 产品线 | 角色 |
|---|---|---|
| 组合一「打工人钱包」 | #1薪资 + #2年终奖 + #4退休 + #5赔偿 + #6社保 + #14调休 + #15工作日 | 主站，pSEO+GEO核心，广告+B端变现 |
| 组合二「报名考试」 | #20报名照 + #22压缩 + #21换底色 + #37文献格式 | 单次付费现金流 |
| 组合三「内容创作者」 | #29降AI味 + #33小红书文案 + #38违禁词 + #30周报 | 订阅制，客单价最高 |
| 组合四「教师」 | #31评语 + #39课堂工具 + 座位表/成绩分析 | B端潜力，小红书教师社群获客 |

**先做组合一或组合三其中一条线跑通闭环，再横向复制。**
