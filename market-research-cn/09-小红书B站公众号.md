# 第十部分：小红书 + B站 + 微信公众号内容策略

## 10.1 平台分工模型

| 平台 | 角色 | 内容形态 | 转化路径 |
|---|---|---|---|
| 小红书 | 场景种草+女性/学生用户 | 图文教程、工具合集、结果晒图 | 笔记→评论区/主页→搜一搜品牌词→小程序 |
| B站 | 深度信任+教程SEO(B站视频在百度权重高) | 3-8分钟实操教程、政策解读 | 简介区链接+置顶评论 |
| 公众号 | 私域承接+微信搜一搜SEO | 长文+文内小程序卡片 | 搜一搜→文章→小程序/网页 |
| 三者关系 | 小红书拉新→B站建立信任→公众号沉淀与变现 | 一次选题三平台改编 | — |

## 10.2 小红书策略

### 账号定位（三选一起步）
1. 「打工人算钱指南」：工资/社保/赔偿计算场景——目标人群与小红书职场话题高度重合。
2. 「考试报名不翻车」：报名照/时间表/材料清单——学生备考人群浓度全网最高。
3. 「教师百宝箱」：评语/课件/班级管理——教师社群是小红书最活跃垂直群体之一。

### 30个可直接开写的选题

| 类型 | 选题示例 |
|---|---|
| 结果晒图型 | 「月薪2万在杭州到手多少|一张图看懂」「我把2027年拼假攻略做成了日历」「AIGC率68%→9%改前改后对比」 |
| 清单合集型 | 「9个免费到离谱的网站|打工人必收」「考试报名照片规格全汇总」「毕业论文全流程工具清单」「教师期末神器5件套」 |
| 避坑型 | 「社保断缴的后果我替你踩过了」「报名照被拒的6个原因」「PDF工具的隐藏收费套路」「签合同前必查的5个条款」 |
| 教程型 | 「3步把照片压到200K以下」「手机就能做电子签名」「身份证加水印保姆级教程」「五险一金自己查缴费记录」 |
| 情绪热点型 | 「原来我要63岁才能退休…」「这周六要上班？调休日历自查」「被裁员当天我做的5件事」「过年亲戚称呼作弊表」 |
| 人设日常型 | 「独立开发第X天：给打工人做了个算钱工具」（开发者日记流，小红书对独立开发人设极友好） |

### 小红书运营要点
- 封面即结论：大字报封面（结果数字/对比图），首图决定80%点击。
- 引流合规：小红书封链接极严——用「评论区自见」「主页瞬间」、企业号可挂小程序；核心是让用户**搜一搜品牌词**，所以工具名要好记、独特。
- 发布节奏：每周3-4篇，同一选题可做「图文版+视频版」双发。
- 爆款翻新：跑出来的笔记每季度换数据重发（小红书流量以新笔记为主）。

### B站要点
- 标题带数字与冲突；封面同小红书大字报逻辑。
- 简介区+置顶评论放网址（B站允许）；视频内口播品牌名≥2次。
- 「独立开发」题材在B站有天然流量池（科技区/编程区UP主生态），开发过程本身就是内容。
- B站视频会被百度收录且排名好——标题按搜索词写（如「五险一金怎么算」而非「算钱那些事」）。

### 公众号要点
- 标题=搜索词本身（搜一搜是精确匹配偏好）；首段直接给答案（微信摘要抓取）。
- 每篇文内嵌小程序卡片（工具直达），文末「工具箱合集」菜单。
- 模板消息/服务号：政策更新提醒（社保调整/放假通知）是最正当的触达理由，打开率高。
- 付费动作放小程序内完成（微信支付无摩擦）。

## 10.6 精准获客的长期机制

1. **工具即内容**：每个计算结果页做「生成分享图」按钮（带品牌水印）——用户晒图=免费投放。
2. **数据即选题**：站内搜索词/计算参数聚合成「打工人工资报告」类原创数据内容，反哺全平台传播（媒体喜欢引用→外链→SEO）。
3. **节点日历驱动**：内容排期跟政策日历（7月社保、10月国考、12月个税/放假）——每年同期素材翻新即可，边际成本递减。
4. **评论区=需求雷达**：所有平台评论区的高频提问，两周内变成新工具或新FAQ页。
